# Security Policy

## Supported versions

The latest released version receives security fixes. Older versions are not maintained.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues, pull requests, or discussions.**

Instead, report them privately through GitHub's [Private Vulnerability Reporting](https://github.com/ProgressPlanner/ai-provider-for-webllm/security/advisories/new), or email **security@progressplanner.com**.

Please include:

- A description of the vulnerability and its impact.
- Steps to reproduce, or a proof of concept.
- Any affected versions you have confirmed.
- Suggested remediation, if you have one.

## What to expect

- We aim to acknowledge your report within **3 business days**.
- We will keep you informed as we investigate and work on a fix.
- Once a fix is released, we are happy to credit you in the advisory unless you prefer to remain anonymous.

Thank you for helping keep this project and its users safe.
