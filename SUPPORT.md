# Getting help

Need a hand with **AI Provider for WebLLM**? Here is where to go.

## Before you ask

- Read the [README](README.md) — it covers installation, requirements, and how the provider works.
- Confirm a plugin or feature providing the **WordPress AI Client** is active. Without it, this plugin stays dormant by design.
- Confirm your browser supports **WebGPU** (recent Chrome or Edge). Inference cannot run without it.

## Questions and usage help

For "how do I…" questions and general usage, please open a [GitHub Discussion](https://github.com/ProgressPlanner/ai-provider-for-webllm/discussions) rather than an issue.

## Bug reports and feature requests

If you have found a bug or want to request a feature, open an issue using the [issue templates](https://github.com/ProgressPlanner/ai-provider-for-webllm/issues/new/choose). Bug reports help most when they include your WordPress version, PHP version, browser, and steps to reproduce.

## Security issues

Do **not** open a public issue for security problems. Follow the process in [SECURITY.md](SECURITY.md).
